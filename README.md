# Boop-api

To run the program node index.js
http://localhost:3000/

http://localhost:3000/books

## API Reference

Api:  https://boop-api.vercel.app

#### Get all items

```http
  GET /api/books
```

#### Get all items by id

```http
  GET /api/books/:id
```

#### POST item

```http
  POST /api/books
```

#### PUT item

```http
  PUT /api/books/:id
```

#### DELETE item

```http
  DELETE /api/books/:id
```
