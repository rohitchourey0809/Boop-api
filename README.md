# Boop-api



## API Reference

Api:  https://boop-api.vercel.app

#### Get all items

```http
  GET /books
```

#### Get all items by id

```http
  GET /books/:id
```

#### POST item

```http
  POST /books
```

#### PUT item

```http
  PUT /books/:id
```

#### DELETE item

```http
  DELETE /books/:id
```
